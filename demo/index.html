Limited # of Words: paper-lipsum demo

It displays a specified number of words, provided by the user using the words attribute. All sentences end with a period, even when they're supposed to end with a comma.

Large # of Words: paper-lipsum demo

The whole paragraph is repeated several number of times if a word count larger than the default count is provided.
